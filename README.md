## 프로젝트 설정 및 실행 방법

### 프로젝트 클론

```
git clone https://github.com/Dev-FE-3/toy-project1-team2
```

### 의존성 설치

```
npm install
```

### 개발 서버 실행

```
npm run server 
```

### 클라이언트 실행

```
npm run dev 
```
